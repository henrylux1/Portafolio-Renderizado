Buscador de imagenes pixabay


Este sitio web es un buscador de imagenes Pixabay que se comunica con una API, cuando colocamos un tema y presionamos sobre buscar imagenes nos vamos a traer del banco de imagenes de Pixabay de la API una serie de imagenes, si le damos click a ver imagen nos lleva a la imagen de alta resolucion y algo importante es que las imagenes Pixabay las podemos utilizar para nuestros proyectos Freelancer.
