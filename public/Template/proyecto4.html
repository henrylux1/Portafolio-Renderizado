Administrador de citas veterinaria


Este sitio es un administrador de citas de pacientes para una veterinaria, donde debemos ingresar los datos de nuestra mascota tales como nombre, tambien debemos ingresar el nombre el propietario y el telefono, tambien se debe ingresar fecha y hora y los sintomas de la mascota. Cuando se ingresen todos los datos ya podemos crear la cita para despues poder visualizarlo en el listado de citas, se pueden editar o eliminar citas.


Este proyecto fue creado con html css y javascript, trabaja con eventos y DOM scripting
