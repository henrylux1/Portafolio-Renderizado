Simulador de Gmail


Este proyecto es un simulador de envio de gmail, nos presenta una interfaz con 3 campos campos para rellenar uno para introducir el gmail de la persona a la que le enviaremos el gamil, otro campo es para el asunto y el ultimo campo para el mensaje. Despues de haber llenado todos los campos ya podemos enviar el gamil si tratamos de enviar sin antes haber llenado al menos un campo no podremos ennviar el gmail, por ultimo tenemos la opcion de resetear el formulario.


En este proyecto fue creado con Html Css y javascript, se trabaja validacion de formularios porque si tratamos en enviar sin antes haber llenado los campos solicitados no vamos a poder enviar el gmail, tambien trabaja con eventos.
